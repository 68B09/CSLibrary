# IPv4
**概要**
==========
IPv4アドレスの扱いをアシストするクラス群です。  
動作に関してはソースファイルも参照して下さい。

●[**IPv4Base**](/doc/IPv4Base.md)
------
基本的なメソッドを収めたクラスです。  

●[**IPv4Range**](/doc/IPv4Range.md)
------
IPv4アドレスの範囲を扱うクラスです。  

●[**IPv4RangeCollection**](/doc/IPv4RangeCollection.md)
------
List&lt;IPv4Range&gt;を継承したクラスです。
